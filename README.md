# Install
`yarn` or `npm i`

## Run
`npm start`

- will build a minified build file
- will start the app on [localhost:3000](http://localhost:3000)

## Dev Run
**In order to run in Development mode:**
`yarn dev` or `npm run dev`
then you can run:
`node .`
_tip: Better run `nodemon .` here_ :+1:
